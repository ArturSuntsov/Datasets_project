# 📋 Kanban-доска для задач

## Обзор

Новый компонент `KanbanBoard` добавляет полноценную поддержку Drag-n-Drop для управления задачами на странице "Задачи".

## ✨ Возможности

- **Drag-n-Drop** — перетаскивание задач между колонками
- **5 статусов** — pending, in_progress, review, completed, rejected
- **Визуальная обратная связь** — подсветка колонки при перетаскивании
- **Цветовая кодировка сложности** — зелёный (лёгкие), жёлтый (средние), красный (сложные)
- **Подсчёт задач** — количество задач в каждой колонке
- **Фильтрация** — фильтрация задач по статусу
- **Прогресс выполнения** — визуальная шкала завершения задач
- **Тёмная тема** — полная поддержка тёмной темы
- **Адаптивность** — корректное отображение на всех устройствах

## 🎨 Как использовать

### Перетаскивание задач

1. Откройте страницу **"Задачи"** (/tasks)
2. Найдите задачу в одной из колонок
3. Зажмите левую кнопку мыши на карточке задачи
4. Перетащите на нужную колонку
5. Отпустите кнопку — статус задачи обновится автоматически

### Фильтрация

Используйте кнопки над доской для фильтрации задач:
- 📋 **Все** — показать все задачи
- 📝 **Ожидает** — только pending задачи
- ⏳ **В работе** — только in_progress задачи
- 👀 **На проверке** — только review задачи
- ✅ **Завершено** — только completed задачи
- ❌ **Отклонено** — только rejected задачи

## 🔧 Технические детали

### Компонент KanbanBoard

**Props:**
```typescript
type KanbanBoardProps = {
  tasks: Task[];
  onStatusChange: (taskId: string, newStatus: TaskStatus) => void;
};
```

**Статусы:**
```typescript
type TaskStatus = 
  | "pending"      // 📝 Ожидает
  | "in_progress"  // ⏳ В работе
  | "review"       // 👀 На проверке
  | "completed"    // ✅ Завершено
  | "rejected";    // ❌ Отклонено
```

### Карточка задачи

Отображает:
- ID задачи (сокращённый)
- Dataset ID
- Исполнитель (если назначен)
- Дедлайн (если установлен)
- Дата создания
- Сложность в процентах с цветовой индикацией

### Drag-n-Drop реализация

Используется нативный HTML5 Drag-and-Drop API:
- `draggable={true}` — делает карточку перетаскиваемой
- `onDragStart` — сохраняет ID перетаскиваемой задачи
- `onDragOver` — подсвечивает колонку
- `onDrop` — вызывает `onStatusChange` с новым статусом

## 🎯 Пример использования

```typescript
import { KanbanBoard } from '../components/KanbanBoard';

function TasksPage() {
  const { data } = useQuery({
    queryKey: ['tasks'],
    queryFn: () => tasksAPI.list({ limit: 100 }),
  });
  
  const updateMutation = useMutation({
    mutationFn: ({ id, status }) => 
      tasksAPI.update(id, { status }),
    onSuccess: () => queryClient.invalidateQueries(['tasks']),
  });
  
  return (
    <KanbanBoard 
      tasks={data?.items ?? []}
      onStatusChange={(taskId, newStatus) => {
        updateMutation.mutate({ id: taskId, status: newStatus });
      }}
    />
  );
}
```
